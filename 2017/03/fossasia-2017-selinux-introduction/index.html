FOSSASIA 2017 - SELinux Introduction talk

Date Sun 19 March 2017 Tags selinux / talks / fossasia

FOSSASIA 2017 was this weekend and I gave an introductory talk on SELinux. The event details are here. There were a lot of great talks over the three days. Engineers.sg filmed all the talks too so you can see them all on the FOSSASIA youtube channel.

In my talk I tried to give a quick overview of SELinux so that people would be more familiar with the basic concepts behind SELinux. I covered DAC vs MAC, SELinux labels and basics of how the rules are layed out. There were lots of great questions too and we had some fun playing around with a little backdoor shell in the webserver and the protections that SELinux had.
